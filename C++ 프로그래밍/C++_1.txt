C(순차적 프로그래밍) + 객체지향 프로그램 => C++

순차적 프로그램 --> 기상 -> 세수 -> 밥 -> 등교 -> 공부 -> 논다 -> 하교 -> 학원
이런식으로 순차적으로 각각 동작이 차례대로 짜내려가는 프로그래밍

객체지향 프로그램 --> C++, JAVA, C# 확장성, 유지보수성이 뛰어남(재활용)

클래스는 붕어빵을 먹고싶다 했을때, 붕어빵 기계와 같다.
(여러개의 붕어빵(객체)으로 이루어져 있는것이 클래스다)

객체는 어떻게 이루어져 있는가? 속성, 동작
--> 객체가 가지고 있는 성격, 학생이 가지고 있는 성질(이메일,학번,이름 등 == Properties)
또한 객체가 가지고 있는 동작들(운동한다. 공부한다 == Method)

<C에서 출력작업>
#include<stdio.h>
=> C언어에서 기본으로 제공하는 입력 출력 함수들을 미리 가지고와서 이 본문과 연결하는 작업
=> 표준 입출력 헤더파일을 여기로 가져오세요. 라는 뜻

Void main() {
	print f("hello c\n"); C의 콘솔로그
}

<C++에서 출력작업>
#include<iostream>
객체지향형언어이므로, iostream라는 클래스(기본 입출력함수)를 가져와서 쓰겠습니다.
int main(void) {
	std::cout<<"hello c"; C++의 콘솔로그, cout (c 아웃)은 객체이다.
	std는 네임스페이스 <<는 연산자와 같은 역할(출력 연산자)
} // 제대로 실행되면 리턴값 0을 반환

계산작업
C언어에서는 결과값을 출력할 때, 서식문자와 함께 써주어야한다.
printf("결과=%d", 결과값), 숫자면 %d, 문자면 %c / -> 서식문자

C++의 표준 입출력을 위한 기본지식
화면에 뭔가를 출력할 때는, 표준 출력스트림(std::cout)를 사용한다.
키보드로 부터 데이터를 입력받을시 표준 입력스트림(std::cin)을 사용한다.
C언어에서 키보드로 부터 데이터를 입력받을때는
(입출력헤더,메인함수 생략)
int i;
scanf("%d, &i") // 서식문자,변수의 주소값
이렇게 쓴다.

클래스의 정의
붕어빵 클래스 = > 붕어빵 객체를 구현하는 역할(구현하는)

Namespace 라는것은 닷넷계열의 언어 C#, VB DOTNET에서도 쓰이는 개념
입출력클래스 스트림에 대한 정보는 Namespace:std std라는 공간에 따로 정의를 했다라고 이해를 하면됨.
std = standard
cin 객체는 >> 연산자를 이용, 입력한다.
cout 객체는 << 연산자를 이용, 출력한다.
